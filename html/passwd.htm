Password amministratore e indirizzo e-mail

Univirt è un sistema GNU/Linux completo e altamente sicuro basato su Debian.

Inserisci la password di root in questo step.


Password: Per favore usa una password complessa. Dovrebbe avere almeno 8 caratteri. Nella password combina simboli, numeri e maiuscole.

E-mail: Inserisci un indirizzo e-mail valido. Il tuo server Univirt invierà avvisi importanti a questo indirizzo (ad esempio errori nei backup, eventi sull'alta disponibilità, etc.).

Premi il pulsante Avanti per proseguire.
