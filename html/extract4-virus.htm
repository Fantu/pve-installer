Univirt cluster

Con Univirt puoi unire differenti server fisici in un unico cluster.

Questo multi-master flessibile elimina i single point of failure (SPOF).

Gestione centralizzata
Crea e gestisci nuovi server virtualizzati semplicemente scegliendo il nodo sul quale avviarli.
Filesystem Cluster (pmxcfs)
Filesystem basato su database per la memorizzazione dei file di configurazione, replicato in tempo reale su tutti i nodi con corosync.
