Gestione della configurazione di rete

Per favore verifica la configurazione di rete mostrata. Avrai bisogno di una configurazione di rete valida per accedere all'interfaccia di gestione dopo l'installazione.

Successivamente premi il pulsante Avanti per proseguire l'installazione. L'installer quindi partizionerà il tuo hard disk e avvierà la copia dei pacchetti.


Indirizzo IP: Imposta l'indirizzo IP per il sistema Univirt.

Netmask: Imposta la netmask della tua rete.

Gateway: Indirizzo di rete del tuo firewall o gateway.

Server DNS: Indirizzo IP del tuo server DNS.
